# Clustering Analysis
Study of different clustering algorithms using different pre-processing techniques with different numbers of clusters on different evaluation parameters


## Dataset Used
- Heart Failure Clinical Records: https://archive.ics.uci.edu/dataset/519/heart+failure+clinical+records
- Number of instances: 299
- Number of Features: 12


## Algorithms
1. K-means Clustering
2. Hierarchical Clustering
3. K-means shift Clustering


## Best Models

After evaluating the models, the following configurations yielded the best results:

- **K-means Clustering**: 
  - **Data Processing**: No data processing
  - **Number of Clusters**: 3

- **Hierarchical Clustering**: 
  - **Data Processing**: No data processing
  - **Number of Clusters**: 5

- **K-means Shift Clustering**: 
  - **Data Processing**: No data preprocessing
  - **Number of Clusters**: 3, 4, and 5
